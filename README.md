## 🏃‍♂️ Running the Portfolio

1. Clone the repository
   ```bash
   git clone https://github.com/Senaraufi/Senaraufi.git
   cd Senaraufi
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```
