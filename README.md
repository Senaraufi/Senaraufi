## 🏃‍♂️ Local Development

1. Clone the repository
   ```bash
   git clone https://github.com/Senaraufi/professional-portfolio.git
   cd professional-portfolio
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Run development server
   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```
